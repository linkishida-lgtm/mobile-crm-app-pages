# 鬼フォロー 公開ページ

このリポジトリは、Google Play / App Store 申請で使う公開ページ専用です。

## 公開予定URL

GitHub Pages を有効化すると、以下のURLで公開されます。

- 案内トップ: https://linkishida-lgtm.github.io/mobile-crm-app-pages/
- サポートURL: https://linkishida-lgtm.github.io/mobile-crm-app-pages/support.html
- プライバシーポリシーURL: https://linkishida-lgtm.github.io/mobile-crm-app-pages/privacy-policy.html

## GitHub Pages 設定

1. このリポジトリの `Settings` を開く。
2. 左メニューの `Pages` を開く。
3. `Build and deployment` の `Source` で `Deploy from a branch` を選ぶ。
4. `Branch` で `main` を選ぶ。
5. フォルダで `/root` を選ぶ。
6. `Save` を押す。
